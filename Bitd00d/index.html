How to play

Combine elements to preform different attacks. Single elements cost 1 coin, while double elemental attacks cost 2. Enemies drop coins. Press 'space' to preform an attack. Explore the different combinations of elements and dont run out of coins!

Up arrow Jump
Left arrow Go left
Right arrow Go right
Q Select fire
W Select water
E Select earth
R Select wind
T Clear element queue
space Cast spell
Esc Start over
